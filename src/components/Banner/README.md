# `Banner`

Renders a general Blueprint Banner component.

## Usage

```jsx
import { Banner } from 'components';

const MyBanner = () => <Banner />;
```

## Props

The `Banner` component accepts a `notificationBanner` prop containing the following fields:

### backgroundColor

Type: `String`
Required: No

### fontColor

Type: `String`
Required: No

### content

Type: `String`
Required: No
